# 👋 简介

嗨。

《问候无疆》（以下简称“该游戏”）是一款基于游戏《吸烟室》的二次创作的同人视觉小说游戏。

该游戏使用 Godot 游戏引擎制作。

该游戏目前仍在制作当中，如果您感兴趣，欢迎关注此项目。

该游戏的首个公开版本预计将于 2023 年 8 月份发布，敬请期待。

感谢您的耐心等待和理解。

# 🧪 如何自行构建？

你可以自行构建这个项目，通过如下步骤：

1. 克隆这个项目。
2. 确保你已经安装了 Godot 游戏引擎 V3.5.2（不是 Mono 版本）。
3. 使用 Godot 启动器打开这个项目。
4. 完成了！如果一切顺利，你将会正常打开本项目在你的本地设备。
